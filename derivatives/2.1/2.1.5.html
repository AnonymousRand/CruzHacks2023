2.1.5 Chain Rule

Introduction

There is one last rule we have yet to learn about, called the chain rule. The chain rule must be invoked whenever a function is a composite function—i.e., it is a function of another function that both contain a variable. You will get better at recognizing these over time, but some examples are:

$y = \sqrt{5x^2 + x} \text{ which is made up of functions } f(x) = \sqrt{x} \text{ and } g(x) = 5x^2 + x$

$y = \sin(\ln(2x^2 + 2)) \text{ which is made up of functions } f(x) = \sin(x),\ g(x) = \ln(x), \text{ and } h(x) = 2x^2 + 2$

The chain rule states that when you encounter a composite function, you must differentiate it as follows:

$y = f(g(x)) \implies y' = f'(g(x)) \cdot g'(x)$

Basically, you take the derivative of the outside function first while leaving the inside function as it is, and then after that you multiply everything by the derivative of the inside function. The chain rule can also be applied multiple times for even more complicated functions:

$y = f(g(h(x))) \implies y' = f'(g(h(x)) \cdot g'(h(x)) \cdot h'(x)$

Any sane exam-maker would usually not require you to apply the chain rule more than twice.

Example Problems

1. Find the derivative of $y = \sqrt[3]{\frac{5}{x} + x}$.

Here we must first recognize that the outside function is $f(x) = \sqrt[3]{x}$ and the inside function is $g(x) = \frac{5}{x} + x$.

The inside function can be rewritten as $g(x) = 5x^{-1} + x$ so we can apply the power rule, and the outside function can be rewritten as $f(x) = x^{\frac{1}{3}}$, giving us $y = (5x^{-1} + x)^{\frac{1}{3}}$.

Now that everything has been optimized for an easy differentiation process, we can go ahead and apply the chain rule: $y' = f'(g(x)) \cdot g'(x) = \frac{1}{3}(5x^{-1} + x)^{-\frac{2}{3}} \cdot (-5x^{-2} + 1)$, and then simplify as needed.

2. Find the derivative of $y = \left(\frac{x + 2}{3x^2} - 3x + 4 \right)^{81423}$.

The outside function is $f(x) = x^{81423}$ and the inside function is $g(x) = \frac{x + 2}{3x^2} - 3x + 4$, which you might notice also involves the quotient rule.

Patiently solving it step-by-step, we end up with $y' = 81423\left(\frac{x + 2}{3x^2} - 3x + 4 \right)^{81422} \cdot \left(\frac{(3x^2)(1) - (x + 2)(6x)}{9x^4} - 3 \right)$, and again simplify as needed.

Practice Problems

1. Find the derivative of $y = 18\sqrt{x^3 + \frac{x}{3}}$

Answer (hover): $y' = 9\left(x^3 + \frac{x}{3} \right)^{-\frac{1}{2}}\left(3x^2 + \frac{1}{3}\right)$

2. Find the derivative of $y = 5x(x^2 + 5x)$

Answer (hover): $y' = 15x^2 + 50x$ (this isn’t actually a chain rule question because you can just distribute the $5x$!)
