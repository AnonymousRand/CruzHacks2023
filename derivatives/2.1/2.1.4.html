2.1.4 Power, Product, & Quotient Rules

Introduction

Now that we’ve gotten the complicated stuff (*wink wink*) out of the way, we can finally move on to the shortcut ways to find a derivative instead of having to use that contrived limit definition. The most basic rule applies to all $x^n$ terms and is called the power rule:

$y = ax^n \implies y' = anx^{n - 1}$

You can apply this to as many added or subtracted terms as you want, as long as they are in the form $x^n$ (yes, the exponent can be negative or fractional too). And if you need to take a derivative of a constant, the derivative of a constant is always 0.

However, once terms involving a variable are multiplied or divided with other non-constant terms, we need to first invoke the product rule and the quotient rule:

$y = f(x)g(x) \implies y' = f'(x)g(x) + f(x)g'(x)$

$y = \frac{f(x)}{g(x)} \implies y' = \frac{g(x)f'(x) - f(x)g'(x)}{(g(x))^2}$

I won’t bother proving these, but the most important thing to keep in mind is that more practice = more natural memorization of these shortcut rules.

Example Problems

1. Find the derivative of $y = \frac{\sqrt{x}}{x + 12}$.

We can rewrite $\sqrt{x}$ as $x^{\frac{1}{2}}$ which would allow us to use the power rule on it.

Then since we have a fraction as the all-encompassing operation, we invoke the quotient rule to get $y = \frac{(x + 12)\left(\frac{1}{2}x^{-\frac{1}{2}} \right) - \left(x^{\frac{1}{2}} \right)(1)}{(x + 12)^2}$, incorporating the power rule whenever necessary as well.

Then we can algebraically simplify this however much we want; the calculus portion of this problem has already been done.

2. Find the derivative of $y = \frac{4}{x^{\frac{2}{3}}}$.

We have another fraction here, but his time since only the denominator contains a variable, we can actually rewrite this function as $4x^{-\frac{2}{3}}$ which will allow us to do everything with the power rule instead of the complicated quotient rule.

Evaluating with power rule, we get $-\frac{8}{3}x^{-\frac{5}{3}}$.

3. Find the derivative of $y = (x + 3)\left(\sqrt{2x} - 4x - 2 \right)$.

You can distribute out the terms and use power rule here, but I’m gonna demo a product rule because eventually you’ll reach a point where you can’t distribute.

Rewrite $\sqrt{2x}$ as $\sqrt{2}x^{\frac{1}{2}}$ for power rule during the product rule.

You should get $(x + 3)\left(\frac{1}{2}\sqrt{2}x^{-\frac{1}{2}} - 4 \right) + (1)\left(\sqrt{2x} - 4x - 2\right)$ (simplify as needed).

4. Solve the limit $\lim_{x \to 3}{\left(\lim_{x \to a}{\frac{(x^2 + 2x) - (a^2 + 2a)}{x - a}} \right)}$.

You might feel a bit intimidated by the double limit, but if you remember the limit definition of a derivative and realize that the inside limit is just the derivative of $x^2 + 2x$ (which is $2x + 2$ using the power rule), you can immediately simplify this problem down to $\lim_{x \to 3}{(2x + 2)}$.

This limit should be pretty easy and evaluates to 8 simply by plugging in 3, since there is no indeterminate form.

Practice Problems

1. Find the derivative of $y = \frac{2x^4}{5}$.

Answer (hover): $y' = \frac{8x^3}{5}$

2. Is the function $y = \frac{9}{x - 4.5}$ differentiable on the interval $(0, 10)$?

Answer (hover): $y' = \left(\frac{1}{3} \right)\left((x^2 + \sqrt{x})(6) + (6x + 2)(2x + \frac{1}{2}x^{-\frac{1}{2}} \right)$ (note that since x is only in one half of the fraction, we can factor out the $\frac{1}{3}$ and then do the product rule)
