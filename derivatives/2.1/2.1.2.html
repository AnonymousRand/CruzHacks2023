2.1.2 Limit Definition of a Derivative

Introduction

While there are many, many shortcuts when it comes to actually finding derivatives, it becomes unfortunately necessary to first consider the “proper”, more tedious way to find a derivative first. We do this through limits (yep, that thing). There are 2 commonly-accepted limit definitions of a derivative (note that we have replaced $y$ with $f(x)$ for more convenient limit notation; they mean the same thing):

$f'(x) = \lim_{h \to 0}{\frac{f(x + h) - f(x)}{h}}$

$f'(x) = \lim_{x \to a}{\frac{f(x) - f(a)}{x - a}}$

These functions may seem daunting, but the first one basically says to take two points on the curve, do a rise/run linear slope calculation between those two points, and then keep narrowing down the distance between the two points. Once we get to an arbitrarily small distance, which the limit is responsible for doing, the points are so close together that the linear slope between them is essentially the same as the slope of the curve between them. The second definition basically says the same thing, except $h$ is replaced with an expression $x - a$ that means the same thing: the distance between the two points. You won't need to know the proofs for these functions.

Example Problems

You might be asked to identify the limit definition of a function’s derivative; basically taking this long way to calculate the derivative:

1. Find the limit definition of the derivative of $y = 2x^2 + 3x - 1$.

To do this, we plug the original function $f(x)$ into either the first or the second form, taking careful consideration of when we need to replace any $x$s with $(x + h)$ in the first form or $a$ in the second.

Its first form derivative is $\lim_{h \to 0}{\frac{(2(x + h)^2 + 3(x + h) - 1) - (2x^2 + 3x - 1)}{h}}$ and its second form derivative is $\lim_{x \to a}{\frac{(2x^2 + 3x - 1) - (2a^2 + 3a - 1)}{x - a}}$.

Keep in mind that you may have to algebraically simplify these fractions using the limit rules from earlier in order to find a matching answer on multiple-choice questions, or just get a more comprehensible derivative function in general.

You might also be tasked with the reverse operation, identifying the original function from its derivative expressed in limit form:

2. Find the function whose derivative is given by $\lim_{h \to 0}{\frac{(2 - (x + h)^2) - (2 - x^2)}{h}}$

Here you notice the presence of $h$ in the limit, so we automatically know that we are dealing with the first form.

To tackle this form, we look at the second half of the numerator, which is then just $f(x)$, the original function. So in this case, the original function is $y = 2 - x^2$.

In the case that we are dealing with the second form instead, which can be detected by the lack of $h$ (or the two terms in the denominator instead of one), we look at the first half of the numerator.

3. Find the function whose derivative is given by $\lim_{x \to w}{\frac{(2x + 2) - (2w + 2)}{w - x}}$.

Don’t let the change from $a$ to $w$ scare you—the form is still functionally the same, and therefore the original function is $y = 2x + 2$.

Practice Problems

1. Find the limit definition of the derivative of $y = \frac{2x}{3}$.

Answer (hover): $\lim_{h \to 0}{\frac{\left(\frac{2(x + h)}{3} \right) - \left(\frac{2x}{3} \right)}{h}}$ or $\lim_{x \to a}{\frac{\left(\frac{2x}{3} \right) - \left(\frac{2a}{3} \right)}{x - a}}$

2. Find the function whose derivative is given by $\lim_{x \to a}{\frac{4\tan{x} - 4\tan{a}}{x - a}}$.

Answer (hover): $y = 4\tan{x}$

3. Find the function whose derivative is given by $\lim_{t \to 0}{\frac{\left(\ln{\left((x + t)^3 \right)} - \frac{1}{(x + t)} \right) - \left(\ln{\left(x^3 \right)} - \frac{1}{x} \right)}{t}}$.

Answer (hover): $y = \ln{\left(x^3 \right)} - \frac{1}{x}$
