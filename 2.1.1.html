Introduction

A derivative is simply a function that represents the slope, or rate of change, of a graph.

For example, we all know from $y = mx + b$ that the slope of $y = 7x + 3$ is $7$. Therefore, the derivative of this function, $y = 7x + 3$, is simply the function $y' = 7$, where the apostrophe denotes a derivative. Similarly, the derivative of the horizontal line $y = 3$ is simply $0$, as there is no rise in the traditional slope formula rise/run.

But what if we want to find the derivative of a curve such as $y = x^2$, instead of a straight line where we have a constant and easily-deducible slope? To do this, we can visualize the slope of a curve at exact, instantaneous points as straight lines that touch the curve briefly:

In the graph above, we’ve drawn a straight line (in white) that briefly touches the curve $y = x^2$ at the point $x = 1$. We call this line the tangent line to $y = x^2$ at the point $x = 1$, and the slope of this tangent line matches the exact slope of the curve at this singular, infinitely small point. We therefore call the slope of a tangent line the instantaneous rate of change, since the slope measures the rate at which y changes. However, we’ve only figured out the slope or derivative of this curve at a singular point $x = 1$. If we want to model every single possible point on a curve with tangent lines to find the derivative, wouldn't we end up with infinite tangent lines? We can see that if we change the point at which the line meets the curve, the slope of the tangent line changes too:

So our goal is to create a general derivative function that outputs all the slopes of the infinite tangent lines as we move the point along the curve. We will cover exactly how to do this later, but for now, the function that models the slopes of all the tangent lines for $y = x^2$ is $y' = 2x$. Therefore, we say that the derivative of $y$ is $y' = 2x$.

How does this derivative work? It allows us to calculate the instantaneous rate of change, or the slope of the tangent line, at any point along the curve. For example, our first tangent line example was at the point $x = 1$. To find the slope of this tangent line (remember: slope of tangent line = slope of curve at that point = instantaneous rate of change of curve at that point), we simply plug in the point $x = 1$ into our derivative function: $2(1) = 2$. Similarly, the slope of this curve at $x = -1$ is $2(-1) = -2$, and the slope of this curve at $x = 0$ is $2(0) = 0$.

Here’s another example: at any point x on the curve $y = 3.5\sqrt[3]{x}$, its slope can be modeled with its derivative function $y' = \frac{3.5}{3} x^{-\frac{2}{3}}$. Again, we will go over how we find these derivatives in a later lesson, but for now, just remember that the derivative models the instantaneous rate of change of a curve at any point along it; i.e, the slope.
